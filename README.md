## 📂 Wybrane Projekty

### 1. Modelowanie Demograficzne i Ekonomiczne (USA 1960-1995)
* **Cel:** Wyjaśnienie i prognozowanie zmian populacji USA w oparciu o wskaźniki makroekonomiczne.
* **Narzędzia:** Gretl[cite: 3].
* **Kluczowe wyniki:** Zbudowano model o bardzo wysokim dopasowaniu ($R^2 = 0.997$)[cite: 3]. Przeprowadzono pełną diagnostykę statystyczną, w tym testy na homoskedastyczność (White'a, Breuscha-Pagana) oraz autokorelację (Durbina-Watsona)[cite: 3].

### 2. Prognozowanie Cen Złota (Szeregi Czasowe)
* **Cel:** Stworzenie modelu predykcyjnego dla kwartalnych cen sprzedaży złota.
* **Narzędzia:** MS Excel (Modele analityczne, wygładzanie wykładnicze Holta)[cite: 5].
* **Kluczowe wyniki:** Model uzyskał 95% wiarygodności prognozy przy średnim błędzie względnym ok. 8,24%[cite: 5].
