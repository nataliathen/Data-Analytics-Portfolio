# Portfolio Analityczne - Natalia Then 📊

Dzięń dobry ! Jestem absolwentką Analityki Biznesowej (obrona: lipiec 2026) z doświadczeniem w analizie danych finansowych i modelowaniu ekonometrycznym. Specjalizuję się w wyciąganiu wniosków biznesowych z danych przy użyciu narzędzi BI oraz metod statystycznych.

## 🛠 Umiejętności techniczne
* **Narzędzia:** MS Excel (zaawansowany), Power BI, Tableau, Gretl.
* **Języki:** SQL (T-SQL), R / R Studio.
* **Metody:** Analiza szeregów czasowych, prognozowanie, regresja liniowa (KMNK).

## 📂 Wybrane Projekty

### 1. Modelowanie Demograficzne i Ekonomiczne (USA 1960-1995)
* **Cel:** Wyjaśnienie i prognozowanie zmian populacji USA w oparciu o wskaźniki makroekonomiczne.
* **Narzędzia:** Gretl.
* **Kluczowe wyniki:** Zbudowano model o bardzo wysokim dopasowaniu (R^2 = 0.997). Przeprowadzono pełną diagnostykę statystyczną, w tym testy na homoskedastyczność (White'a, Breuscha-Pagana) oraz autokorelację (Durbina-Watsona).

### 2. Prognozowanie Cen Złota (Szeregi Czasowe)
* **Cel:** Stworzenie modelu predykcyjnego dla kwartalnych cen sprzedaży złota.
* **Narzędzia:** MS Excel (Modele analityczne, wygładzanie wykładnicze Holta).
* **Kluczowe wyniki:** Model uzyskał 95% wiarygodności prognozy przy średnim błędzie względnym ok. 8,24%.
